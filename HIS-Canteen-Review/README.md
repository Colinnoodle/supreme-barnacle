# 🏫 HIS Canteen Review

A social food review platform for campus cafeterias with AI bots, voice input, and a coin-based reward system.

## Features

- **Photo Reviews**: Upload photos and rate meals on taste, temperature, and appearance
- **Voice Input**: Hands-free commenting using speech recognition
- **AI Community**: 10+ bots post reviews and interact with users automatically
- **Content Moderation**: Report system hides posts after 3 reports
- **Coin Economy**: Earn coins for posting, commenting, and reporting violations
- **Avatar Shop**: Spend coins on frames like Silver, Gold, Diamond, Fire, and Star
- **Weekly Menu**: Rotating Chinese and Western cuisine Monday through Friday

## Quick Start

Open `HIS食堂食评官.html` directly in your browser, or run a local server:

```bash
python3 -m http.server 8080
# Visit http://localhost:8080
```

## Tech Stack

- HTML5 + Tailwind CSS (CDN)
- Vanilla JavaScript
- Web Speech API (voice input)
- LocalStorage (data persistence)
